# Портфолио Дизайнера Штор

Этот проект представляет собой сайт-портфолио.

## Как запустить сайт

1.  **Открыть терминал**:
    *   В VS Code нажмите `Ctrl + ~` (тильда), чтобы открыть встроенный терминал.
    *   Или откройте командную строку (cmd/PowerShell) и перейдите в папку проекта: `cd c:\Users\User\Documents\Проекты\www`

2.  **Установить зависимости (если еще не установлены)**:
    ```bash
    npm install
    ```

3.  **Запустить сервер разработки**:
    ```bash
    npm run dev
    ```

4.  **Открыть сайт**:
    *   После запуска вы увидите ссылку в терминале, обычно это [http://localhost:5173/](http://localhost:5173/).
    *   Нажмите `Ctrl + Click` по ссылке или скопируйте её в браузер.

## Структура проекта

*   `index.html` - Главная страница.
*   `src/main.js` - Логика сайта и данные портфолио (список проектов).
*   `src/styles/main.css` - Стили сайта.
*   `public/images/` - Папка для изображений.

## Как добавить новый проект

1.  Добавьте фотографии в папку `public/images/`.
2.  Откройте файл `src/main.js`.
3.  Найдите массив `projects` и добавьте новый объект с данными проекта (id, название, описание, список фото).
